Who We Are

At Ultra Things we make websites and the software that powers them, everything from web front-end work (the pretty bits that you see) to back-end work (the powerful bits that you don't see).

We carry out contract work for large projects and are available for hire directly for new projects.

We make things with:

  • HTML5 and CSS3

  • Responsive layouts

  • PHP

  • Symfony2 framework

  • JavaScript and JQuery

  • Twig and liquid templating

We're comfortable with:

  • Server-side JavaScript (Node)

  • Ruby and Ruby on Rails

  • Groovy and Grails

  • Java

  • Bash scripting

We power things with:

  • Debian-based Linux distros

  • Apache

  • Nginx

  • PHP-FPM

  • Jekyll
